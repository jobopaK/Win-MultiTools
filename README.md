# Win-MultiTools

Herramienta multipropósito en PowerShell para gestión de Red y Sistema.

## Estructura

- `app/`: Lógica principal del programa.
  - `src/`: Scripts fuente.
  - `tests/`: Pruebas de los scripts.
  - `build.ps1`: Script para construir el ejecutable final `Win-MultiTools.ps1`.
- `docs/`: Documentación del proyecto.
